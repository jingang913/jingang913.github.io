hexo 3.X 安装配置

Hexo 是一个轻量的静态博客框架。通过Hexo可以快速生成一个静态博客框架,仅需要几条命令就可以完成,相当方便。

而架设Hexo的环境更简单了 不需要lnmp/lamp这些繁琐复杂的环境 仅仅需要一个简单的http服务器即可使用 或者使用互联网上免费的页面托管服务

下面是架设Hexo3的环境步骤：

1、在github上新建一个repositories，名字为username.github.io

2、配置ssh，将公钥配置到github上

3、安装hexo

4、新建项目，在blog目录下

()内为缩写形式
上传代码步骤：
(1)hexo clean
(2)hexo g
(3)hexo d

5、修改配置_config.yml

6、一些常用命令：
hexo new “postName” #新建文章
hexo new page “pageName” #新建页面
hexo generate #生成静态页面至public目录
hexo server #开启预览访问端口（默认端口4000）
hexo deploy #将.deploy目录部署到GitHub
hexo help # 查看帮助
hexo version #查看Hexo的版本

7、若报错ERROR Deployer not found: git 或者 ERROR Deployer not found: github

8、修改页面小图标
修改主题下的_config.yml

并将favicon.ico放在主目录下resource里
